# FULL-SCREEN VIDEO EXAMPLE

## Summary

This example displays a video covering the width and height of the screen, on desktop and any orientation on mobile.

The video HEIGHT extends to the full height of the screen. The video WIDTH is cropped to fit, and content is CENTERRED.

When the screen loads, the video autoplays on a loop, without audio.

## Usage

Replace the sample video file with a video following the specs below. 
Replace or remove the sample text in the HTML.

## Video Specs

* 5 to 10 seconds duration, despigned to play on a loop
* 5MB or smaller
* no audio in the file (NOTE: this is to optimize performance; audio won't play, regardless)
* mp4 file format
